
# MIPR Daily Puzzle

## Description
MIPR Daily Puzzle is a daily guessing game where players try to guess a PR (Power Ranking) player based on a series of hints. Each day, a new puzzle is generated, and players have a limited number of guesses to identify the correct player.

## How to Play
1. Guess the PR player of the day.
2. Start with the PR season and unlock additional hints with each guess.
3. You have a maximum of 4 guesses to identify the correct player.
4. After each guess, you will receive feedback indicating whether the guessed player's rank is higher or lower than the correct player's rank.

## Random Game Function
1. You will be given a random puzzle with the same rules.
2. Try to get as many in a row as you can.
